# TLS1on1

TLS1on1 is a online skill exchange website. It is a web application that users who can exchange skills and knowledge.

## How to use?

```bash
# check
sh run.sh doctor

# start frontend
cd frontend
npm i
npm run dev

# start backend
cd backend
docker build -t tls1on1-back
docker compose up
```
